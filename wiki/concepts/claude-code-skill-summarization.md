# Claude Code Skill Summarization

## Overview

Claude Code Skill Summarization is a process where AutoSkills generates a summary file (CLAUDE.md) listing the installed AI agent skills relevant to Claude Code workflows. This summary provides a quick reference for both human developers and AI agents, facilitating skill discovery and invocation.

## How It Works

When AutoSkills detects that a project targets Claude Code—either automatically or via the `-a` flag—it creates a `CLAUDE.md` file in the project root. This file is generated by aggregating information from the installed markdown skill files located in `.claude/skills`.

The summarization process scans the `.claude/skills` directory, collecting metadata and references for each skill. For example, it lists accessibility skills, Astro usage guides, deployment skills, design thinking patterns, Node.js backend patterns, and more. Each skill entry includes references to its main markdown file and any supplementary files, such as pattern libraries or configuration guides.

The summary is structured to provide actionable context: it describes when to use each skill, what triggers its invocation (e.g., "improve accessibility", "deploy my app"), and where to find detailed documentation. For skills with multiple references, the summary links to pattern files, configuration guides, and advanced usage notes.

This approach enables both developers and AI agents to quickly understand the available capabilities in the project. It also supports agentic workflows by making skill invocation explicit and discoverable. The summary is regenerated automatically whenever skills are installed or updated, ensuring it remains current with the project's state.

The CLAUDE.md file is especially useful in multi-agent environments, where different agents may need to reference or invoke skills based on project requirements. It acts as a living index of agentic capabilities, supporting both manual and automated workflows.

## Key Properties

- **Automatic Generation:** CLAUDE.md is generated automatically when Claude Code is detected or specified.
- **Skill Indexing:** Lists all installed skills and references, providing actionable context for invocation.
- **Agentic Workflow Support:** Facilitates skill discovery and invocation for both human and AI agents.

## Limitations

Only available when Claude Code is detected or specified. The summary depends on the accuracy of skill detection and may not cover custom or manually added skills. It is limited to the skills installed via AutoSkills and may not reflect external capabilities.

## Example

Excerpt from CLAUDE.md:

```
## Accessibility (a11y)
Audit and improve web accessibility following WCAG 2.2 guidelines. Use when asked to "improve accessibility", "a11y audit", "WCAG compliance", "screen reader support", "keyboard navigation", or "make accessible".
- `.claude/skills/accessibility/SKILL.md`
- `.claude/skills/accessibility/references/A11Y-PATTERNS.md`: Practical, copy-paste-ready patterns for common accessibility requirements.
- `.claude/skills/accessibility/references/WCAG.md`
```
