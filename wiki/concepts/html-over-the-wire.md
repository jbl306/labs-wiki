# HTML Over The Wire

## Overview

HTML Over The Wire refers to the approach of using HTML attributes to enable dynamic web functionality, such as AJAX requests and WebSockets, directly within HTML elements.

## How It Works

HTML Over The Wire utilizes custom attributes like `hx-post` and `hx-swap` to define behavior directly in HTML. These attributes trigger actions such as HTTP requests or DOM updates based on user interactions, simplifying the development process by reducing reliance on external JavaScript frameworks.

## Key Properties

- **Attribute-Based:** Uses HTML attributes to define dynamic behavior.
- **Dependency-Free:** Does not require external libraries or frameworks.
- **Extendable:** Supports extensions for additional functionality.

## Relationship to Other Concepts

- **[[AJAX]]** — htmx integrates AJAX functionality into HTML.

## Practical Applications

Used for building modern user interfaces with minimal JavaScript, enabling dynamic content updates and interactions.
