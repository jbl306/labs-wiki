# Query Operation in LLM Wiki

## Overview

The query operation allows users to ask questions against the LLM Wiki and receive synthesized answers with citations.

## How It Works

When a user poses a question, the LLM searches the wiki's index for relevant pages, reads the content, and generates a synthesized answer. The answer can be saved back into the wiki as a new page, allowing the knowledge base to grow and evolve with each query.

## Key Properties

- **Citations:** Answers include citations to relevant wiki pages or sources.
- **Dynamic Formats:** Answers can be presented in various formats, such as markdown pages, tables, or charts.
- **Knowledge Compounding:** Answers can be saved back into the wiki to enrich the knowledge base.

## Relationship to Other Concepts

- **[[LLM Wiki]]** — Querying is a key operation in the LLM Wiki system.

## Practical Applications

Used to retrieve and synthesize information from the wiki for research, decision-making, or learning.
