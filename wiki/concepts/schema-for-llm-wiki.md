# Schema for LLM Wiki

## Overview

The schema is a configuration document that defines the structure, conventions, and workflows for the LLM Wiki. It ensures the LLM acts as a disciplined wiki maintainer.

## How It Works

The schema specifies how the wiki is organized, including the format of pages, cross-referencing conventions, and the process for ingesting sources, answering queries, and maintaining the wiki. It evolves over time as the user and LLM refine their workflow.

## Key Properties

- **Customization:** The schema is tailored to the user's domain and preferences.
- **Guidance:** It provides rules and workflows for the LLM to follow during operations.
- **Evolvability:** The schema can be updated as the user's needs and the wiki evolve.

## Relationship to Other Concepts

- **[[LLM Wiki]]** — The schema is a core component of the LLM Wiki architecture.

## Practical Applications

Used to guide the LLM in maintaining a structured and consistent knowledge base.
