# AppWorld Benchmark

## Overview

AppWorld is a suite of autonomous agent tasks involving API understanding, code generation, and environment interaction. It provides a realistic execution environment with common applications and APIs, and tracks performance via a public leaderboard.

## Relevance

AppWorld is used as the primary agent benchmark in the paper, demonstrating ACE's superior performance and scalability compared to strong baselines. It highlights the difficulty and realism of agent tasks requiring multi-turn reasoning and tool use.
