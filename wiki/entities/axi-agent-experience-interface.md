# AXI (Agent eXperience Interface)

## Overview

AXI is a framework and set of ergonomic standards for building CLI tools optimized for AI agent interaction. It defines 10 principles that prioritize token efficiency, usability, and structured output, validated by benchmark studies and reference implementations.

## Relevance

AXI addresses the inefficiencies of traditional CLI and MCP protocols, enabling agents to operate with higher accuracy and lower token cost. It is a foundational framework for agent-centric automation.
