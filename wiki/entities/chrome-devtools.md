# Chrome DevTools

## Overview

Chrome DevTools is a suite of developer tools built into the Chrome browser for debugging, inspecting, and analyzing web applications. Chrome DevTools MCP exposes its API to coding agents via the MCP protocol, enabling deep inspection and performance analysis.
