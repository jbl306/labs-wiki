# DeepSeek-V3.1

## Overview

DeepSeek-V3.1 is an open-source large language model used as the backbone for ACE's Generator, Reflector, and Curator components in the paper's experiments. It enables fair evaluation by preventing knowledge transfer from stronger models.

## Relevance

DeepSeek-V3.1 is the base LLM for ACE's experiments, ensuring fairness and isolating the benefit of context construction. ACE achieves superior performance using DeepSeek-V3.1 compared to production-level agents powered by GPT-4.1.
