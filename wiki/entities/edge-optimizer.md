# Edge Optimizer

## Overview

Edge Optimizer is a module that performs threshold optimization for betting edges using walk-forward cross-validation and applies the Kelly criterion for fractional bet sizing. It is designed to improve betting strategy profitability and risk management in the NBA ML engine.
