# fxtwitter API

## Overview

An unofficial API endpoint for Twitter/X content that returns tweet data in JSON format without requiring authentication. It supports fetching tweet text, author metadata, media URLs, and quoted tweets. It is used to bypass limitations of scraping Twitter's JavaScript-rendered pages and enables programmatic access to tweet content.
