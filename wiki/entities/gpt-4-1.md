# GPT-4.1

## Overview

A large multimodal language model accessible via GitHub Models API, supporting both text and image inputs. GPT-4.1 offers significantly higher throughput than GPT-4o and includes vision capabilities enabling interpretation of base64-encoded images alongside text.
