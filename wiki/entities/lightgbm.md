# LightGBM

## Overview

LightGBM is an open-source, high-performance gradient boosting framework developed by Microsoft. It is designed for efficiency, scalability, and accuracy, particularly with large datasets, and supports advanced tree learning techniques.

## Key Facts

| Field | Value |
|-------|-------|
| Type | Framework |
| Created | 2016 |
| Creator | Microsoft |
| URL | https://github.com/microsoft/LightGBM |
| Status | Active |

## Relevance

LightGBM is widely used in machine learning for classification, regression, and ranking tasks, offering superior speed and memory efficiency compared to other boosting frameworks.

## Associated Concepts

- **Leaf-Wise Tree Growth** — Core tree construction strategy in LightGBM.
- **[[Gradient-Based One-Side Sampling (GOSS)]]** — Sampling innovation for efficient training.
- **Histogram-Based Learning** — Optimization for fast, memory-efficient tree construction.
- **Feature Importance and SHAP Values** — Integrated interpretability tools.
