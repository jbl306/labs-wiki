# puppeteer

## Overview

Puppeteer is a Node.js library that provides a high-level API to control Chrome or Chromium over the DevTools Protocol. It is used by Chrome DevTools MCP for reliable browser automation, enabling actions such as navigation, input simulation, and screenshot capture.
