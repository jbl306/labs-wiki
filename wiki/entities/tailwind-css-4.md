# Tailwind CSS 4

## Overview

Tailwind CSS 4 is a utility-first CSS framework used in the NBA ML Engine React dashboard redesign. It is configured with the @tailwindcss/vite plugin and uses the `@theme` directive for defining design tokens as CSS custom properties, enabling light/dark theming and consistent styling.
