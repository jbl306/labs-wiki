# Vectorize

## Overview

Vectorize is a Cloudflare primitive providing vector search over embedded memories in Agent Memory. It enables semantic retrieval by indexing memory embeddings and supporting upserts and deletions for superseded memories.

## Relevance

Vectorize allows Agent Memory to perform semantic search, supporting hybrid retrieval pipelines and efficient recall of relevant memories. It maintains vector indices for each memory profile, enabling scalable, isolated retrieval.
