# Workers AI

## Overview

Workers AI is a Cloudflare primitive for running LLMs and embedding models used in Agent Memory. It supports local model inference for extraction, verification, classification, and synthesis tasks within the memory pipeline.

## Relevance

Workers AI enables Agent Memory to run extraction, classification, and synthesis pipelines efficiently on Cloudflare's network. It supports session affinity for prompt caching and allows flexible model selection for different pipeline stages.
