# Agentic Context Engineering: Evolving Contexts for Self-Improving Language Models

## Summary

This paper introduces ACE (Agentic Context Engineering), a framework for evolving and optimizing contexts in large language model (LLM) applications. ACE addresses the limitations of brevity bias and context collapse in prior context adaptation methods by treating contexts as comprehensive, structured playbooks that are incrementally updated and refined. The framework demonstrates significant performance gains and efficiency improvements across agent and domain-specific benchmarks, enabling scalable, self-improving LLM systems.

## Key Points

- ACE prevents context collapse and brevity bias by incrementally updating contexts with structured, itemized entries.
- ACE outperforms strong baselines in agent and domain-specific tasks, with notable gains in accuracy and reduced adaptation latency.
- ACE enables self-improving LLMs without labeled supervision, leveraging execution feedback and environment signals.
