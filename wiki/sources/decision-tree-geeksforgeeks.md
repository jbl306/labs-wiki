# Decision Tree - GeeksforGeeks

## Summary

This article introduces Decision Trees, explaining their structure, types, working mechanism, splitting criteria, pruning, advantages, disadvantages, and practical applications. It provides a clear overview of how Decision Trees are used for classification and regression tasks in machine learning, highlighting their interpretability and versatility. The article also discusses common pitfalls like overfitting and computational challenges, and includes visual diagrams to illustrate the concepts.

## Key Points

- Decision Trees are used for classification and regression tasks in machine learning.
- Key concepts include tree structure, splitting criteria (Gini impurity, entropy), and pruning.
- Advantages include interpretability and handling non-linear relationships; disadvantages include overfitting and instability.

## Concepts Extracted

- **[[Decision Tree Algorithm]]** — A Decision Tree is a supervised learning algorithm used for both classification and regression tasks. It models decisions and their possible consequences in a tree-like structure, making it easy to interpret and visualize. Decision Trees are widely used in machine learning due to their versatility and ability to handle both categorical and numerical data.
- **[[Pruning In Decision Trees]]** — Pruning is a technique used to reduce the complexity of Decision Trees and prevent overfitting. By removing branches that have little predictive power, pruning helps the model generalize better to unseen data and improves performance.
