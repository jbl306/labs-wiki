# htmx - HTML Over The Wire

## Summary

htmx is a lightweight JavaScript library that enables rich, modern user interfaces by allowing HTML elements to directly perform AJAX, CSS transitions, WebSockets, and Server Sent Events through declarative attributes. The project emphasizes simplicity, extensibility, and minimal dependencies, aiming to 'complete' HTML as a hypertext by removing arbitrary constraints on HTTP requests and UI updates.

## Key Points

- htmx lets HTML elements perform HTTP requests and UI updates using attributes, not imperative JavaScript.
- It supports AJAX, CSS transitions, WebSockets, and Server Sent Events natively in HTML.
- The library is small (~14k min.gz'd), dependency-free, and highly extensible via extensions.
