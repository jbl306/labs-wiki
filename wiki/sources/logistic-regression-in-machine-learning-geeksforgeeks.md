# Logistic Regression in Machine Learning - GeeksforGeeks

## Summary

This article provides a comprehensive overview of logistic regression, a supervised classification algorithm in machine learning. It explains the mathematical formulation, types, assumptions, implementation, evaluation metrics, and contrasts logistic regression with linear regression. The guide includes code examples and visualizations to illustrate key concepts.

## Key Points

- Logistic regression predicts probabilities for classification, not continuous values.
- It uses the sigmoid function to map linear outputs to probabilities between 0 and 1.
- There are binomial, multinomial, and ordinal variants, each suited to different types of categorical targets.
- Maximum likelihood estimation is used to fit model parameters.
- Evaluation metrics include accuracy, precision, recall, F1 score, AUC-ROC, and AUC-PR.

## Concepts Extracted

- **[[Logistic Regression]]** — Logistic regression is a supervised machine learning algorithm designed for classification tasks. It models the probability that an input belongs to a particular class, using a linear combination of input features transformed by the sigmoid function. Unlike linear regression, which predicts continuous values, logistic regression outputs probabilities for categorical outcomes.
- **[[Sigmoid Function]]** — The sigmoid function is a mathematical function used in logistic regression to map any real-valued input into a probability between 0 and 1. Its S-shaped curve makes it ideal for converting linear model outputs into interpretable probabilities for classification tasks.
- **[[Maximum Likelihood Estimation in Logistic Regression]]** — Maximum likelihood estimation (MLE) is the method used to fit the parameters of logistic regression. It seeks the weights and bias that maximize the likelihood of observing the given data, ensuring the model predicts probabilities that best match the observed outcomes.
