# Logistic Regression in Machine Learning - GeeksforGeeks

## Summary

This article provides a comprehensive overview of logistic regression, a supervised machine learning algorithm used for classification tasks. It covers the mathematical foundations, types, assumptions, implementation details, evaluation metrics, and contrasts logistic regression with linear regression. Practical Python code examples for both binomial and multinomial logistic regression are included, along with explanations of key concepts such as the sigmoid and softmax functions.

## Key Points

- Logistic regression is used for classification, not regression, and predicts probabilities for categorical outcomes.
- The sigmoid function transforms linear outputs into probabilities between 0 and 1.
- Maximum likelihood estimation is used to fit the model, and evaluation metrics include accuracy, precision, recall, F1 score, and AUC.

## Concepts Extracted

- **[[Logistic Regression]]** — Logistic regression is a supervised learning algorithm designed for classification tasks, particularly binary classification. It models the probability that a given input belongs to a specific class by applying a sigmoid function to a linear combination of input features. Unlike linear regression, which predicts continuous values, logistic regression outputs probabilities, making it suitable for problems where the target variable is categorical.
- **[[Sigmoid Function]]** — The sigmoid function is a mathematical function that maps any real-valued input into the range (0, 1). It is central to logistic regression, where it converts the linear predictor into a probability, enabling binary classification.
- **[[Maximum Likelihood Estimation in Logistic Regression]]** — Maximum likelihood estimation (MLE) is the method used to fit the parameters of logistic regression. It seeks to find the weights and bias that maximize the probability of observing the given data, ensuring the model's predictions align closely with the actual outcomes.
